使用说明

📝 如何添加内容？

  1. 在对应的标签页中填写表单
  2. 点击"添加"或"发布"按钮
  3. 切换到"导出数据"标签页
  4. 点击"生成JSON代码"按钮
  5. 复制生成的代码
  6. 将代码粘贴到项目的 data/content.json 文件中
  7. 保存文件并刷新网站

💡 注意事项

  • 所有数据都保存在浏览器本地存储中
  • 定期导出备份数据，避免丢失
  • 图片需要单独上传到服务器，然后填写URL
  • 标签用逗号分隔，不要有空格

🚀 GitHub部署流程

  1. 将所有文件上传到GitHub仓库
  2. 在仓库设置中启用 GitHub Pages
  3. 选择 main 分支作为源
  4. 等待几分钟后访问你的网站
  5. 每次更新内容后，提交到GitHub即可自动更新
